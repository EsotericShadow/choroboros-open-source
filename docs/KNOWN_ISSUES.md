# Known Issues (Beta)

Issues known for Choroboros 2.01-beta. Please report additional issues via the Feedback button.

## Beta Testers

- **Platform:** macOS only (VST3, AU, Standalone). Windows and Linux builds not yet available.
- **Dev Panel:** Developer panel is visible (About > Dev Panel or similar). Intended for power users; some controls may be experimental.

## Audio / DSP

- _None currently documented. Please report any clicks, pops, or unexpected behavior._

## UI / UX

- _None currently documented. Please report layout issues, tooltip errors, or confusing controls._

## Compatibility

- **DAWs:** Tested with Reaper. Other DAWs (Logic, Ableton, etc.) should work but please report any host-specific issues.
- **Sample rates:** Supports up to 192 kHz. Report any issues at extreme sample rates.

## Reporting Issues

Use the **Feedback** button in the plugin to submit issues via our Google Form. Usage statistics can be saved to a file and pasted into the form.
